### 🚀 5 Minutes to Live Deployment (No DevOps Required!)

- Setting this up on your account takes just a few small, straightforward steps:

- Download & Customize: Grab the source code and drop your personal details into the configuration files.

- Push to GitHub: Upload the code to your personal private repository.

- Configure 2 Lines: Update your GitHub repository paths in the workflow file.

- Secure Access: Add a standard GitHub Access Token as a repository secret so the workflow can securely talk to your hosting repo.

- Set Version Variables: Add 2 simple variables in your repository dashboard to control the automatic cache-busting versioning.
